# Security Policy

Time Translator handles user calendar data, Jira credentials, and payment information via Stripe. If you find a security vulnerability, please report it responsibly.

## Reporting a vulnerability

**Do not open a public GitHub issue.** Instead, email **<contact@timetranslator.com.au>** with:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs, screenshots, or proof-of-concept code

We aim to acknowledge reports within 3 business days.

## Scope

In scope: the application at [timetranslator.com.au](https://timetranslator.com.au) and this repository.

Out of scope: third-party services we integrate with (Google, Atlassian, Stripe, Supabase, Upstash, Anthropic) — please report issues in those services directly to their respective vendors.
